imgCentral

滑动分页组件

移动端，通过手式向上滑动，切换下一页内容。

使用说明

名字 类型 默认值 说明
配置参数
width String "100%" 图片宽度，支持"\d+px" "\d+%" "auto"
height String "100%" 图片高度，支持"\d+px" "\d+%" "auto"
src String null 图片地址
display String "block" 支持'inline','block','inline-block',目前仅测试了'block'
vertical-align String "middle" 支持'top','bottom','middle',目前仅测试了'middle'
text-align String "center" 支持'left','right','center',目前仅测试了'center'
