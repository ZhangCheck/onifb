imgCentral

图片组件

图片布局是前端常用功能，也是难点功能，该组件旨在简化图片布局方式。

使用说明

名字 类型 默认值 说明
配置参数
tip String "加载中..." 加载内容提示信息
numPerPage Number 10 页大小，每页内容的条目数
interface Object null 接口配置
pageField String "page" 接口发出请求时，上传的数据的字段名（当前页码数据）
numPerPageField String "rows" 接口发出请求时，上传的数据的字段名（页大小数据）
tipFixBottom Number 0 提示信息距离底边的距离
