textBoxDialog

使用说明

名字 类型 默认值 说明
配置参数
id String 不写则随机产生 因为一个页面中可能会出现多个textBoxDialog为确保中的textbox,dialog,以及button按钮的vmodelId是唯一的。 此值得产生是根据comboBox的vmodelId值，若comboBox的vmodelId为空，者自动产生一个值
iconPosition String right 用于设定弹出按钮的位置，值有"left","right"
width String 200 textBoxDialog的宽度值
onOpen function 弹出窗口打开窗口时执行的方法
onConfirm function 弹出窗口点击确定执行的方法
onClose function 关闭弹出窗口执行的方法
onCancel function 取消弹出窗口执行的方法
title String 弹出窗口的标题
fileTemplateUrl String 窗口内容的html路径值，窗口中的内容是在另一个文件中渲染完成数据，路径中只能是html代码，且不含有js部分（e8环境,ms-include-src引入外模板的时候,外模板页面上的js不运行,ie9及高级点的浏览器就能运行），该文件中的js渲染在textBoxDialog中执行
fileTemplate String 窗口内容的html字符串，窗口中的内容是在另一个文件中渲染完成数据
selectedValue String textbox的对应默认值
selectedCode String textbox值对应的code值
