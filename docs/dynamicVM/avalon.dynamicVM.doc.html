fb.load(url,selectorOrElem,[callback(title)])

使用说明

功能：将url指定的目标页面加载到当前页面的一个元素中

  1. 第一个参数传入的url应当是文件相对于require根路径的路径
  2. 第二个参数为选择器字符串或DOM对象，在IE6及不支持querySelector的浏览器中请使用id选择或直接传入DOM对象
  3. 第二个参数为选择器字符串是只对与之匹配的第一个元素进行操作
  4. 被加载的子页面body元素会被去除所以不要把ms-controller写在body中

参数列表

名字 类型 默认值 说明
url string "" 被夹载页面的url
selectorOrElem string/HTMLElement null DOM对象或选择器
callback(title) Function 加载完成回调(title为被加载页面title标签的内容)
