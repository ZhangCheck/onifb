comboBox

使用说明

表单中参数的每一个表单参数名称都必须以opts结尾（不区分大小写）,

名字 类型 默认值 说明
配置参数
column int 3 一行显示的表单的列数
titleWidth int 100 表单标题的宽度
*opts {} eg:NameTextOpts opts前面的字符串在同一个表单中必须是一个唯一的值
type:String类型 渲染的类型 "textBox" 为文本框，"dropDown"下拉选择
"date"时间，"textBoxDialog"文本弹出框
"comboBox" 可输入可选择的选择框
name:String类型 显示在表单左侧的名称
opts：{} 表单参数eg: <input ms-widget="textbox, a, $aOpts"/>中的$aOpts值
如：dropDownOpts:{//参数名名称必须以Opts结束 type:"dropDown",//显示的类型 name:"下拉选择", opts:{ container:"dropB", listWidth:"100%", titleWidth:"90%", data: [ { value: "选择1", label: "选择1" }, { value: "选择2", label: "选择2" } ] } }
