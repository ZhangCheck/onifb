mmAnimate

动画引擎

基于单时间轴的动画引擎

使用方法

 avalon(elem).animate(properties[, duration][, easing][, complete])
 avalon(elem).animate(properties, options)
Frame (关键帧)
名字 类型 默认值 说明
elem Element 处于动画状态的元素节点
$events Object {} 放置各种回调
troops Array [] 当queue为true，同一个元素产生的帧对象会放在这里
tweens Array [] 放置各种补间动画Tween
orig Object {} 保存动画之前的样式，用于在隐藏后还原
dataShow Object {} 保存元素在显示时的各种尺寸，用于在显示前还原
bind(type, fn, unshift)
参数名/返回值 类型 说明
type String 事件名
fn Function 回调，this为元素节点
unshift Undefined|String 判定是插在最前还是最后
fire(type, [otherArgs..]) 触发回调，可以传N多参数
build() 控制样式，显示元素
createTweens() 创建补间动画
revertTweens() 反转补间动画数组
Tween 补间动画
名字 类型 默认值 说明
elem Element 处于动画状态的元素节点
prop Object {} 当前时间的css样式
easing function avalon.easing["linear"] 缓动公式
run(per, end) 更新元素的某一样式或属性，内部调用
cur() 取得当前值
原型方法
名字 类型 默认值 说明
delay() 延时
pause() 暂停
resume() 继续
stop(clearQueue,gotoEnd)
参数名 类型 说明
clearQueue boolean 清除动画队列为true的话等待执行的帧都不会执行
gotoEnd boolean 是否跳跃到当前一帧的终结状态

clearQueue的值 gotoEnd的值 效果
true true 立即完成该元素的所有动画
true false 清空该元素的所有动画
false true 立即跳到最后一帧，继续下一个动画
false false 中断当前动画，继续下一个动画

一个时间轴 avalon.timeline中包含许多帧, 一帧里面有各种渐变动画, 渐变的轨迹是由缓动公式所规定

绑定属性ms-animate 参数配置使用说明
名字 类型 说明
options Object 动画参数配置
名字 类型 说明
duration number 动画持续时间
easing array 缓动公示
fps number 动画每秒帧数
before,step,after function 在动画开始前、每一个动作后、完成后执行的方法
properties Object 目标状态样式配置 与css样式相同,目前支持数值渐变和颜色渐变
  • 算子 && fps 修改 with && backgroundColor
  • 算子 && fps slideToggle, slideUp, slideDown
  • 算子 && fps fadeToggle, fadeIn, fadeOut
  • 算子 && fps toggle, show, hide
  • ms-animate属性绑定
  • ms-animate与ms-if,ms-visible结合使用
