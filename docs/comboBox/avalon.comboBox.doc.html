comboBox

使用说明

输入框中的值可以通过手输入，或下拉选择，改变值

名字 类型 默认值 说明
配置参数
width int 200 textBoxDialog的宽度值
selWidth String 100% 选择框的默认宽度
id String 不写则随机产生 因为一个页面中可能会出现多个comoboBox为确保中的textbox以及button按钮的vmodelId是唯一的。 此值得产生是根据comboBox的vmodelId值，若comboBox的vmodelId为空，者自动产生一个值
list Array 选择框中的内容 eg:[{"code":"01","value":"选择1"}, {"code":"02","value":"选择2"}, {"code":"03","value":"选择3"}, {"code":"04","value":"选择4"}, {"code":"05","value":"选择5"}, {"code":"06","value":"选择6"}, {"code":"07","value":"选择7"}]
selectedValue String textbox的对应默认值
selectedCode String textbox值对应的code值
listNumber Int 6 选项框中的超过listNumber的个数时，出现滚动条
search Boolean true 是否开启搜索选择类目
nodataText String 无 无数据时显示文本
